# TikTok Video Scripts — accepted.fyi

## How to Create These Videos (CapCut + AI Voice)

### Step 1: AI Voice
- Open CapCut → New Project
- Tap "Text" → paste the voiceover script
- Tap "Text to Speech" → choose a voice (recommended: "Jesse" or "Matt" for authoritative tone, or "Aria" for friendly)
- Adjust speed to ~1.1x for TikTok pacing

### Step 2: Text on Screen
- Add each text card as a separate text layer
- Use bold white text, large font (CapCut → "Montserrat Bold" or "Inter Bold")
- Background: use the dark slate (#0f172a) or black
- Time each card to match the voiceover (2-3 seconds per card)

### Step 3: Styling
- Add subtle zoom animations on each text card (CapCut → Animations → "Fade Up" or "Scale")
- Add background music: search "cinematic" or "dramatic" in CapCut sounds (keep at 15-20% volume)
- Add the accepted.fyi logo as a small watermark in corner

### Step 4: Export
- Export at 1080x1920 (9:16 vertical)
- Use "Export" button (NOT "Share to TikTok") to avoid CapCut watermark
- Save to camera roll, then upload to TikTok separately

### Posting Tips
- Post between 6-9 PM EST (peak student/parent time)
- Add 3-5 hashtags: #collegeadmissions #classof2030 #collegedecisions #acceptedfyi #admissionsdata
- Add a trending sound at low volume under the AI voice for algorithm boost
